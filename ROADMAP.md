# Roadmap

The release sequence for Calibrated Explanations is defined and managed in
`development/current-work/RELEASE_PLAN_v1.md`. That file is the authoritative
source for milestone goals, release gates, and sequencing decisions.

`ROADMAP.md` is an entry-point pointer only — it does not enumerate versions or
milestone summaries. Version-specific details belong in milestone plan files
(`development/current-work/vX.Y.Z_plan.md`) and the release plan, not here.

## Where to look

| What you need | Where to find it |
|---|---|
| Current milestone goals and release gates | `development/current-work/RELEASE_PLAN_v1.md` |
| Active milestone task breakdown | `development/current-work/vX.Y.Z_plan.md` |
| Completed milestone records | `development/finished-work/` |
| ADRs governing architecture | `development/adrs/` |
| Engineering standards | `development/standards/` |
| Development documentation map | `development/README.md` |
