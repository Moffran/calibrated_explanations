# Support

Thanks for using Calibrated Explanations! Here are the best ways to get help:

- **Documentation**: Start at https://calibrated-explanations.readthedocs.io/
  for guides, API references, and troubleshooting steps.
- **Questions & discussion**: Use GitHub Discussions:
  https://github.com/Moffran/calibrated_explanations/discussions
- **Bug reports & feature requests**: Open an issue with the appropriate
  template: https://github.com/Moffran/calibrated_explanations/issues
- **Security issues**: Follow `SECURITY.md` for private reporting channels.

We aim to respond as quickly as possible, but support is best-effort and handled
by maintainers and community volunteers.
