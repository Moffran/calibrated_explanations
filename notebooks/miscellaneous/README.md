# Miscellaneous notebooks

This folder contains supplementary notebooks that are useful once you’ve completed the main learning path in `../README.md`.

## What’s here

- [demo_speeddating.ipynb](./demo_speeddating.ipynb)
  - End-to-end example on the SpeedDating dataset.
- [imbalanced_wrap.ipynb](./imbalanced_wrap.ipynb)
  - Wrapper API usage on imbalanced classification problems.
- [LIME_comparison.ipynb](./LIME_comparison.ipynb)
  - Side-by-side comparisons with LIME-style explanations.
- [support_xgboost_lightgbm_catboost_issue_92.ipynb](./support_xgboost_lightgbm_catboost_issue_92.ipynb)
  - Repro/support notebook for gradient-boosting libraries (may require extra dependencies).

## Generated outputs

Some notebooks may create tool-specific output folders (e.g., `catboost_info/`). These are not part of the conceptual examples; they’re training logs/artifacts produced by the underlying libraries.
