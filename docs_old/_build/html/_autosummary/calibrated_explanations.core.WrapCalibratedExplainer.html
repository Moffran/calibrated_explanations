calibrate(x_calibration: Any, y_calibration: Any, mc: Callable[[Any], Any] | MondrianCategorizer | None = None, **kwargs: Any) → WrapCalibratedExplainer[source]¶

Calibrate the explainer with calibration data.

Parameters:

  • x_calibration (array-like) – The calibration input samples.

  • y_calibration (array-like) – The calibration target values.

  • mc (callable or MondrianCategorizer, optional) – Optional Mondrian categories helper. Defaults to None.

  • **kwargs – Keyword arguments to be passed to the CalibratedExplainer’s __init__ method

Raises:

NotFittedError – If the learner is not fitted before calibration.:

Returns:

The WrapCalibratedExplainer object with explainer initialized as a CalibratedExplainer.

Return type:

WrapCalibratedExplainer

Examples

Calibrate the learner to the calibration data:

w.calibrate(x_calibration, y_calibration)

Provide additional keyword arguments to the CalibratedExplainer:

w.calibrate(x_calibration, y_calibration, feature_names=feature_names,
            categorical_features=categorical_features)

Notes

if mode is not explicitly set, it is automatically determined based on the the absence or presence of a predict_proba method in the learner.
